# Str-calculator
Str-calculator is a calculator, but you input string, and it output string
```
1+1 => 2
sqrt(2)+pi => √2 + π ≈ 4.555806215962888
cos 5 => cos(5) ≈ 0.28366218546322625
1/5 => 1/5 = 0.2
1/3 => 1/3 ≈ 0.3333333333333333
e^(pi*i) => -1
```
And much more
